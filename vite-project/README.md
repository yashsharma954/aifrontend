# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some Oxlint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](httpss://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react) uses [Oxc](httpss://oxc.rs)
- [@vitejs/plugin-react-swc](httpss://github.com/vitejs/vite-plugin-react/blob/main/packages/plugin-react-swc) uses [SWC](httpss://swc.rs/)

## React Compiler

The React Compiler is not enabled on this template because of its impact on dev & build performances. To add it, see [this documentation](httpss://react.dev/learn/react-compiler/installation).

## Expanding the Oxlint configuration

If you are developing a production application, we recommend using TypeScript with type-aware lint rules enabled. Check out the [TS template](httpss://github.com/vitejs/vite/tree/main/packages/create-vite/template-react-ts) for information on how to integrate TypeScript and Oxlint's TypeScript related rules in your project.
